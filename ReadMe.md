Projeto de Árvore binária contígua

Aluno: André Ricardo Cândido Silva

Processos:

- Inserção: ocorre como a passagem da chave a ser inserida e do pai desse nó.
- Remoção: ocorre com a passagem da chave a ser removida.
- Busca: Passa o elemento a ser encontrado. Ele devolve se o elemento encontra-se ou não na lista.
- Impressão: É permitido a impressão através das três formas que se percorre uma árvore binária: a em ordem, a pré ordem e a pós ordem.
- getElementos(): devolve a quantidade de elementos inseridos na árvore.
